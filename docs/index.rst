kosu: Build and maintain courses
================================

    | ``kosu`` is a command-line utlity to help you build and maintain multi-day courses from content in Jupyter Notebooks.


Quick start
-----------

.. toctree::
    :caption: Quick start

Install ``kosu`` with pip:

.. code-block:: shell

    pip install kosu

``kosu`` is a command-line utility. To start a course or collection of courses, do this on the command line:

.. code-block:: shell

    mkdir mycourses
    cd mycourses
    kosu --init

This will create several directories and files in the directory mycourses.
You can run some of the other commands in this guide to see how they work.
